== HLT API

A blog API that allows the user to create, update, and list blog posts and categories. Features include HTTP
token authentication, versioning, and pagination. Gems include Rails 4.1, Active Model Serializers, Kaminari,
and RSpec.

== Getting Started

To install:

1) git clone git@github.com:crestcode/hlt-api.git

2) cd hlt-api

3) bundle install

4) rake db:migrate

5) rspec
